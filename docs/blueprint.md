# **App Name**: DevSphere

## Core Features:

- Dynamic & Interactive Hero Section: Implement a full-screen hero section featuring 'Amit Naganuri', '2nd Year Computer Science Engineering Student', and a tagline with an animated typing effect. Includes interactive navigation buttons for 'View Projects' and 'Contact Me' that smoothly scroll to respective sections.
- Structured Content Presentation: Display detailed 'About' information, animated 'Skills' cards, 'Project' cards (like the 'AI Folder Organizer'), 'How It Works' section, and 'Contact' links in clearly defined and modular sections throughout the portfolio.
- Responsive Layout & Navigation: Ensure the entire portfolio is fully responsive for optimal viewing on desktop and mobile screens (down to 375px), utilizes smooth scrolling navigation, and prevents horizontal scrolling.
- Engaging Animations & Interactions: Apply smooth fade-in animations on scroll for various content sections, implement hover effects with slight lift and soft shadows for skill cards and buttons, and a glowing effect for the 'View on GitHub' button.
- AI Project Description Polisher Tool: Provide an internal tool that uses generative AI to help the user refine and professionalize their project descriptions, optimizing them for clarity and impact on the portfolio.

## Style Guidelines:

- Primary Color: A vibrant yet deep blue (#5164E0) for key interactive elements and accents, representing professionalism and a modern aesthetic.
- Background Color: A very dark, desaturated navy (#1E222D) providing a clean, deep backdrop for content, inspired by a modern blue to dark navy gradient.
- Accent Color: A bright, clear sky blue (#7CD0EC) for highlighting interactive elements, links, and creating dynamic contrast.
- Cards and content containers will feature clean white backgrounds for optimal readability against the dark page background, embodying the 'clean white cards' concept.
- Headline Font: 'Space Grotesk' (sans-serif) for titles like 'Amit Naganuri' and section headers, providing a modern and tech-infused feel. Body Font: 'Inter' (sans-serif) for all other textual content, ensuring excellent readability and a contemporary appearance.
- Use modern, minimalist vector icons for social media links (GitHub, LinkedIn) and potentially to categorize skills, maintaining a professional and clean look.
- The layout will be constructed using Flexbox or Grid for optimal responsiveness. The hero section will be full-screen height. Content will be organized vertically with clear section delineations, preventing horizontal scrolling. The necessary <meta name="viewport" content="width=device-width, initial-scale=1.0"> tag will be included for mobile optimization.
- All interactive elements, including buttons, cards, and navigation, will feature smooth transition effects. Content sections will gradually fade into view as the user scrolls, while interactive components like skill cards and project links will have subtle lift and glowing hover effects.
